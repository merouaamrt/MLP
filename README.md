# MLP
L’objectif est la mise en œuvre d’un (MLP) à l’aide de la bibliothèque FANN

J'ai travaillé sur trois problèmes de complexité croissante : l'apprentissage du XOR, la détection binaire de langues, et enfin la classification multi-langues. J'ai également réalisé l'exercice facultatif sur la classification des fleurs d'iris.
